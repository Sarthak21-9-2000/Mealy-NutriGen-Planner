# 🍽️ AI Meal Planner (NutriGen Planner)

An AI-powered web application that generates personalized 7-day Indian meal plans based on user inputs like age, weight, height, diet preference, and fitness goals.

---

## 🌐 Live Demo

👉 https://mealy-nutrigen-planner.onrender.com/

---

## 🚀 Features

- 🧠 AI-generated personalized meal plans
- 📅 Structured 7-day diet plan
- 🔥 Daily calorie tracking per meal
- 📊 Total daily calorie calculation
- 🎯 Goal-based recommendations (Weight Loss / Gain / Maintain)
- 🥗 Vegetarian & Non-Vegetarian options
- 💻 Clean, modern UI with card layout

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, Bootstrap
- **Backend:** Flask (Python)
- **AI Integration:** OpenAI API
- **Deployment:** Render
- **Server:** Gunicorn
